# Project Page Template

This is a static research project page template intended for GitHub Pages.

## Files to edit

- `index.html`: title, authors, buttons, section text
- `assets/css/style.css`: colors and layout
- `assets/js/main.js`: small interactions

## Replace placeholders

1. Replace the teaser block with your teaser image or video.
2. Replace the overview and method placeholders with your figures.
3. Replace the result placeholders with your demo videos.
4. Update the BibTeX block.

## Publish on GitHub Pages

1. Create a GitHub repo, for example `your-project`.
2. Upload this folder's contents to the repo root.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Your site will be published at `https://YOUR_USERNAME.github.io/your-project/`.
